控制类名（className 属性）

className 属性设置或返回元素的class 属性。

语法：

object.className = classname
    作用:
  1. 获取元素的class 属性
  2. 为网页内的某个元素指定一个css样式来更改该元素的外观
看看下面代码，获得 <p> 元素的 class 属性和改变className： 
    <style>
      .one{
        width: 200px;
        background-color: #ccc;
      }
      .two{
        font-size: 18px;
        color: #F00;
      }
    </style>

    <script type="text/javascript">
      var isOne = true;
      function myFun(){
        var oP = document.getElementById('con');
        if(isOne){
          oP.className = "two";
        }else{
          oP.className = "one";
        }
        isOne = !isOne;
      }
    </script>

    <input type="button" id="btn" value="点击按钮查看效果" onclick="myFun();">
    <p id="con" class="one">点击按钮切换样式

结果:

点击按钮切换样式

插入节点appendChild()

在指定节点的最后一个子节点列表之后添加一个新的子节点。

语法:

appendChild(newnode)

参数:newnode：指定追加的节点。

我们来看看，div标签内创建一个新的 P 标签，代码如下:

    <div id="test">
      <p>HTML</p>
      <p>JavaScript</p>
    </div>
    <script>
      var oDiv = document.getElementById('test');
      var newnode = document.createElement("p");
      newnode.innerHTML = "This is a new p";
      // 通过appendChild()添加节点
      oDiv.appendChild(newnode);
    </script>

运行结果:

HTML

JavaScript

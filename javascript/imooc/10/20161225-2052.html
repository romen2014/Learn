替换元素节点replaceChild()

replaceChild 实现子节点(对象)的替换。返回被替换对象的引用。

语法：

node.replaceChild (newnode,oldnew )

参数：newnode : 必需，用于替换 oldnew 的对象。

参数：oldnew : 必需，被 newnode 替换的对象。

我们来看看下面的代码:

    <button onclick="replaceMessage()">点击按钮 将Apple替换为Banana</button>
    <h4 id="old">Apple</h4>
    <script>
      function replaceMessage(){
        var newnode = document.createElement("h4");
        newnode.innerHTML = "Banana";
        var oldnode = document.getElementById('old');
        oldnode.parentNode.replaceChild(newnode, oldnode);
      }
    </script>

运行结果:

Apple

注意: 1. 当 oldnode 被替换时，所有与之相关的属性内容都将被移除。

注意: 2. newnode 必须先被建立。

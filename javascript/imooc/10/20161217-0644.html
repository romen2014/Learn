内容选中事件onselect

选中事件，当文本框或者文本域中的文字被选中时，触发onselect事件，同时调用的程序就会被执行。

如下代码,当选中用户文本框内的文字时，触发onselect 事件，并调用函数message()。

      <html>
        <head>
          <script type="text/javascript">
            function message(){
              alert("已触发选中事件");
            }
          </script>
        </head>
        <body>
          <form>
            <input id="username" type="text" value="请选中我" onselect="message()" />
          </form>
        </body>
      </html>

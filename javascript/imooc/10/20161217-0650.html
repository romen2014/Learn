文本框内容改变事件onchange

通过改变文本框的内容来触发onchange事件，同时执行被调用的程序。

如下代码,当用户将文本框内的文字改变后，弹出对话框“您改变了文本内容！”。

      <html>
        <head>
          <script type="text/javascript">
            function message(){
              var oTxt = document.getElementById('username');
              var oP = document.getElementById('now');
              oP.innerHTML = "变更后的内容为：" + oTxt.value;
            }
          </script>
        </head>
        <body>
          <form>
            <input id="username" type="text" value="" onchange="message()" />
            <p id="now"></p>
          </form>
        </body>
      </html>

Document Object Model

父子节点

		parentNode	-父节点
		offsetParent	-获取元素在页面上的实际位置(同CSS定位的父级)
		childNodes	-子节点(含文本节点与元素节点交替)
		children	-元素子节点

首尾子节点

		firstChild	-第一子节点(有兼容问题)
		fitstElementChild	-第一个元素节点(有兼容问题，高级浏览器支持)
		lastChild	-最后一个子节点(有兼容问题)
		lastElementChild	-最后一个元素节点(有兼容问题，高级浏览器支持)

兄弟节点

		nextSibling	-下一个同级节点(有兼容问题)
		nextElementSibling	-下一个同级元素节点(有兼容问题，高级浏览器支持)
		previousSibling	-上一个同级节点(有兼容问题)
		previousElementSibling	-上一个同级元素节点(有兼容问题，高级浏览器支持)

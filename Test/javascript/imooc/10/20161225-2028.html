删除节点removeChild()

removeChild() 方法从子节点列表中删除某个节点。如删除成功，此方法可返回被删除的节点，如失败，则返回 NULL。

语法:

nodeObject.removeChild(node)

参数:node ：必需，指定需要删除的节点。

我们来看看下面代码，删除子点。

    <div id="test">
      <p>HTML</p>
      <p>JavaScript</p>
    </div>
    <script>
      var oDiv = document.getElementById('test');
      var oP = oDiv.removeChild(oDiv.childNodes[1]);
      document.write("删除节点的内容为：" + oP.innerHTML);
    </script>

运行结果:

HTML

JavaScript

注意:把删除的子节点赋值给 oP，这个子节点不在DOM树中，但是还存在内存中，可通过 oP 操作。

如果要完全删除对象，给 oP 赋 null 值，代码如下:

    var oP = oDiv.removeChild(oDiv.childNodes[1]);
    oP = null;

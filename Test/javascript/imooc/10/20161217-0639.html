失去焦点事件onblur

onblur事件与onfocus是相对事件，当光标离开当前获得聚焦对象的时候，触发onblur事件，同时执行被调用的程序。

如下代码, 网页中有用户和密码两个文本框。当前光标在用户文本框内时（即焦点在文本框），在光标离开该文本框后（即失焦时），触发onblur事件，并调用函数message()。

      <html>
        <head>
          <script type="text/javascript">
            function message(){
              var oTxt = document.getElementById('username');
              alert("文本框已失去焦点，目前内容为：" + oTxt.value);
            }
          </script>
        </head>
        <body>
          <form>
            <input id="username" type="text" value="" onblur="message()" />
          </form>
        </body>
      </html>

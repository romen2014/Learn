继续循环continue

continue的作用是仅仅跳过本次循环，而整个循环体继续执行。

语句结构：

    for(初始条件;判断条件;循环后条件值更新){
      if(特殊情况){
        continue;
      }
      循环代码
    }

上面的循环中，当特殊情况发生的时候，本次循环将被跳过，而后续的循环则不会受到影响。好比输出10个数字，如果数字为5就不输出了。

    var num;
    for(num = 1; num <= 10; num++){
      if(num==5){
        continue;
      }
      document.write("数值：" + num + "<br />")
    }

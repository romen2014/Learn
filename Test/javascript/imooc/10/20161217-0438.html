do...while循环

do while结构的基本原理和while结构是基本相同的，但是它保证循环体至少被执行一次。因为它是先执行代码，后判断条件，如果条件为真，继续循环。

do...while语句结构：

    do{
      循环语句
    }
    while(判断条件)

我们试着输出5个数字。

    num= 1;
    do{
      document.write("数值为:" +  num+"<br />");
      num++; //更新条件
    }
    while (num<=5)

执行结果:

为什么呢?我们来看下执行思路：

    • 第1次
    • 输出：数值为1
    • 条件值更新：num++，更新为2
    • 判断：2<=5，条件成立，继续进行do...while循环
    • 第2次
    • 输出：数值为2
    • 条件值更新：num++，更新为3
    • 判断：3<=5，条件成立，继续进行do...while循环
    • 第3次
    • 输出：数值为3
    • 条件值更新：num++，更新为4
    • 判断：4<=5，条件成立，继续进行do...while循环
    • 第4次
    • 输出：数值为4
    • 条件值更新：num++，更新为5
    • 判断：5<=5，条件成立，继续进行do...while循环
    • 第5次
    • 输出：数值为5
    • 条件值更新：num++，更新为6
    • 判断：6<=5，条件不成立，结束do...while循环

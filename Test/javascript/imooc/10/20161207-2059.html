自加一，自减一 ( ++和--)

算术操作符除了(+、-、*、/)外，还有两个非常常用的操作符，自加一“++”；自减一“--”。首先来看一个例子：

    mynum = 10;
    mynum++; //mynum的值变为11
    mynum--; //mynum的值又变回10

上面的例子中，mynum++使mynum值在原基础上增加1，mynum--使mynum在原基础上减去1,其实也可以写成:

    mynum = mynum + 1;//等同于mynum++
    mynum = mynum - 1;//等同于mynum--

示例：

    var numa=1;
    var numb=7;
    numa++;
    numb--;
    document.write("numa的值:"+numa);
    document.write("numb的值:"+numb);

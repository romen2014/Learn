焦点事件onfocus

当网页中的对象获得聚点时，执行onfocus调用的程序就会被执行。

如下代码, 当将光标移到文本框内时，即焦点在文本框内，触发onfocus 事件，并调用函数message()。

      <html>
        <head>
          <script type="text/javascript">
            function message(){
              var oTxt = document.getElementById('username');
              oTxt.focus();    //获取焦点
              oTxt.value = "已获得焦点";
              oTxt.select();  //选中已有内容，可直接输入相关信息替换选中的内容
            }
          </script>
        </head>
        <body>
          <form>
            <input id="username" type="text" value="" onfocus="message()" />
          </form>
        </body>
      </html>

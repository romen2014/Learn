页面卸载事件onunload

当用户退出页面时（页面关闭、页面刷新等），触发onUnload事件，同时执行被调用的程序。

注意：不同浏览器对onunload事件支持不同。

如下代码,当退出页面时，弹出对话框“您确定离开该网页吗？”。

      <html>
        <head>
          <script type="text/javascript">
            window.onunload = function message(){
              alert("页面即将关闭");
            }
          </script>
        </head>
        <body>
        </body>
      </html>

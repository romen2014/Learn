页面加载事件onload

事件会在页面加载完成后，立即发生，同时执行被调用的程序。

注意：1. 加载页面时，触发onload事件，事件写在<body>标签内。

注意：2. 此节的加载页面，可理解为打开一个新页面时。

如下代码,当加载一个新页面时，弹出对话框“加载中，请稍等…”。

      <html>
        <head>
          <script type="text/javascript">
            function message(){
              alert("加载中，请稍后...");
            }
          </script>
        </head>
        <body onload="message()">
        </body>
      </html>

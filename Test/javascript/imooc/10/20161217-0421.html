for循环

很多事情不只是做一次，要重复做。如打印10份试卷，每次打印一份，重复这个动作，直到打印完成。这些事情，我们使用循环语句来完成，循环语句，就是重复执行一段代码。

for语句结构：

    for(初始化变量;循环条件;循环迭代){
      环语句
    }

假如，一个盒子里有6个球，我们每次取一个，重复从盒中取出球，直到球取完为止。

    var num=1;
    for (num=1;num<=6;num++){  //初始化值；循环条件；循环后条件值更新
      document.write("取出第"+num+"个球<br />");
    }

执行结果：

执行思路:

    • 第一次
    • 初始化：num=1
    • 判断：num<=6(num值此时为1)
    • 输出：取出第1个球
    • 条件值更新：num++(num=num+1,num值更新为2)
    • 第二次
    • 判断：num<=6(num值此时为2,所以判断2<=6,条件成立)
    • 输出：取出第2个球
    • 条件值更新：num++(num=num+1,num值更新为3)
    • 第三次
    • 判断：num<=6(num值此时为3,所以判断3<=6,条件成立)
    • 输出：取出第3个球
    • 条件值更新：num++(num=num+1,num值更新为4)
    • 第四次、第五次 //中间环节一样，略...
    • 第六次
    • 判断：num<=6(num值此时为6,所以判断6<=6,条件成立)
    • 输出：取出第6个球
    • 条件值更新：num++(num=num+1,num值更新为7)
    • 第七次
    • 判断：num<=6(num值此时为7,所以判断7<=6,条件不成立)
    • 不再执行循环体内的输出语句
    • 结束for循环

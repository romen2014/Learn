鼠标移出事件onmouseout

鼠标移开事件，当鼠标移开当前对象时，执行onmouseout调用的程序。

当把鼠标移动到"登录"按钮上，然后再移开时，触发onmouseout事件，调用函数message()，代码如下:

      <html>
        <head>
          <script type="text/javascript">
            function message(){
              alert("鼠标已移出按钮！");
            }
          </script>
        </head>
        <body>
          <form>
            <input name="button" type="button" value="确定" onmouseout="message()" />
          </form>
        </body>
      </html>

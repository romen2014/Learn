JQuery 可见筛选选择器

元素有显示状态与隐藏状态，jQuery根据元素的状态扩展了可见性筛选选择器:visible与:hidden。

选择器 描述
$(':visible') 选择所有显示的元素。
$(':hidden') 选择所有隐藏的元素。

这2个选择器都是 jQuery 延伸出来的，看起来比较简单，但是元素可见性依赖于适用的样式。

:hidden选择器，不仅仅包含样式是display="none"的元素，还包括隐藏表单、visibility等等。

    我们有几种方式可以隐藏一个元素：
  1. CSS display的值是none。JQuery视为不可见，因为该元素不占用空间布局。
  2. type="hidden"的表单元素。JQuery视为不可见，因为该元素不占用空间布局。
  3. 宽度和高度都显式设置为0。JQuery视为不可见，因为该元素不占用空间布局。
  4. 一个祖先元素是隐藏的，该元素是不会在页面上显示。JQuery视为不可见，因为该元素不占用空间布局。
  5. CSS visibility(可见属性)的值是hidden。JQuery视为可见，因为该元素占用空间布局。
  6. CSS opacity(透明度)的值是0。JQuery视为可见，因为该元素占用空间布局。

如果元素中占据文档中一定的空间,元素被认为是可见的。可见元素的宽度或高度，是大于零。元素的visibility:hidden或opacity:0被认为是可见的，因为他们仍然占用空间布局。
